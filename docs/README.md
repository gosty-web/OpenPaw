# OpenPaw Documentation

Welcome to the official documentation for **OpenPaw**, the open-source platform for running, managing, and creating autonomous AI agents.

OpenPaw provides an incredibly flexible, extensible, and locally-first environment designed to build AI applications. From multi-modal interactions with web searching and browser crawling, to rich tool invocation via MCP servers, OpenPaw is your AI agent hub.

## Core Principles

1. **Local First**: Your agents run on your hardware. Data, memories, and configurations are saved locally unless explicitly exported.
2. **Modularity**: Every capability (Browser, Search, Voice, MCPs) is built as an independent, extensible module.
3. **Agent Anatomy**: We believe agents should have state, memories, and an evolving persona, instead of just stateless API calls.
